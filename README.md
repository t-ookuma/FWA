# How to Setup

`git clone https://github.com/taku300/FWA.git`  
`cd FWA`  
`composer install`  
`vendor/bin/sail up -d --build`  
`cp .env.local .env`  
`vendor/bin/sail artisan migrate --seed`  
`vendor/bin/sail npm install`  
`vendor/bin/sail npm run dev` このコマンドを実行した状態で開発を進める

`localhost:3000` でアクセス
